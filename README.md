# 🍽️ Restaurant App

A desktop **restaurant menu application** built with Python Tkinter as a 1st-semester CS project at Nile University.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-blue)
![Status](https://img.shields.io/badge/Status-Complete-brightgreen)

---

## ✨ Features

- 🏠 **Home Page** — Welcome screen with restaurant introduction
- 🍕 **Menu Page** — Full menu with Egyptian pound pricing
- 👥 **About Page** — Team information
- 🧭 **Navigation Bar** — Smooth page switching between screens

## 🚀 How to Run

**Requirements:** Python 3.x (Tkinter is built-in)

```bash
# Clone the repository
git clone https://github.com/AbdallaYoussef006/restaurant-app.git

# Navigate to project
cd restaurant-app

# Run the app
python mypython.py
```
